# La classe virtuelle de Madame Lanquetin

Site statique prêt pour GitHub Pages.

## Mise en ligne
1. Créez un nouveau dépôt GitHub, par exemple `classe-virtuelle-madame-lanquetin`.
2. Déposez **le contenu de ce dossier** à la racine du dépôt.
3. Dans **Settings > Pages**, choisissez `Deploy from a branch`, branche `main`, dossier `/root`.
4. Activez **Enforce HTTPS** si l’option apparaît.
5. Ouvrez l’adresse fournie par GitHub.

## Confidentialité
- aucun compte, formulaire, outil statistique, publicité ou traceur social ;
- aucune donnée personnelle envoyée par le site ;
- progression et préférences conservées uniquement dans le navigateur (`localStorage`) ;
- images, scripts, styles et carte servis depuis le dépôt ;
- les ressources externes ne sont contactées qu’après un clic explicite ;
- page dédiée : `confidentialite.html` ;
- GitHub Pages enregistre techniquement l’adresse IP des visiteurs pour la sécurité de son service, comme indiqué dans sa documentation.

## Accessibilité
- navigation au clavier et focus visible ;
- lien d’évitement ;
- réglages de taille, contraste, espacement, liens soulignés, réduction des animations et mode concentration ;
- descriptions textuelles des cartes ;
- lecture vocale lorsque le navigateur le permet ;
- page dédiée : `accessibilite.html` ;
- statut déclaré honnêtement : conformité RGAA non évaluée tant qu’un audit complet n’a pas été réalisé.

## Contenu
- 6 périodes historiques ;
- galerie de personnages ;
- frises et cartes schématiques interactives ;
- vocabulaire en cartes retournables ;
- quiz corrigés ;
- exercices sur des unes de presse et une affiche de 1936 ;
- méthodologie d’analyse documentaire ;
- ressources institutionnelles ouvertes seulement après un clic ;
- progression locale exportable ;
- installation possible comme application et fonctionnement hors connexion après une première visite.
